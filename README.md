# Usage 

1. Download repository

        # wget --no-check-certificate http://github.com/tdlmatias/lamp-setup/tarball/master -O lamp-setup.tar.gz

2. Unzip

        # tar xvf lamp-setup.tar.gz
        # cd kbond-lamp-setup-{hash}

3. Run!

        # sh server-setup.sh {name} {email}
